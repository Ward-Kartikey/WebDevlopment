Course Curriculum

Welcome to the Web Development course. This curriculum is designed to provide a comprehensive overview of essential topics. Please review the modules below.

Modules Overview

  • HTML Basics
  • CSS Fundamentals
  • JavaScript Essentials
  • Project Work

Course Objectives & Outcomes

  1. Understand the structure of web pages.
  2. Apply styles using CSS.
  3. Develop interactive features with JavaScript.
  4. Build a complete website as a final project.

Module Details

HTML Basics
Learn about tags, elements, attributes, and document structure.
CSS Fundamentals
Explore selectors, properties, and layout techniques.
JavaScript Essentials
Understand variables, functions, events, and DOM manipulation.
Project Work
Apply all concepts to build a real-world project.

Topics Covered

  • HTML
    1. Headings & Paragraphs
    2. Lists

      • Unordered Lists
      • Description Lists

  • CSS

    • Box Model

  • JavaScript

    • Functions
    • Events
    • DOM Manipulation
